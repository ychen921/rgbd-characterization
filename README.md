# Orbbec ROS2 Workspace

This workspace contains the Orbbec ROS2 driver, Docker environment, DDS communication configuration, and local experiment folders for Orbbec RGB-D camera development.

Current workspace path:

```bash
~/dev/orbbec_ws
```

---


### Folder Purpose

| Folder | Purpose |
|---|---|
| `src/OrbbecSDK_ROS2/` | Orbbec ROS2 wrapper source code |
| `config/` | DDS / ROS2 communication configuration |
| `docker/` | Docker image definition |
| `scripts/` | Host/container launch and environment scripts |
| `bags/` | Local rosbag records |
| `data/` | Local experiment input data |
| `results/` | Local experiment outputs, figures, reports |
| `docs/` | Notes and documentation |
| `build/` | Generated by `colcon build` |
| `install/` | Generated by `colcon build` |
| `log/` | Generated by `colcon build` |

`build/`, `install/`, and `log/` are generated folders and should not be committed to Git.

## Environment Setup

Before using ROS2 commands on the host machine, source the workspace environment script:

```bash
cd ~/dev/orbbec_ws
source scripts/host_orbbec_env.sh
```

This script should configure the ROS2 and DDS environment.

Expected content of `scripts/host_orbbec_env.sh`:

```bash
#!/usr/bin/env bash

source /opt/ros/humble/setup.bash

if [ -f ~/dev/orbbec_ws/install/setup.bash ]; then
  source ~/dev/orbbec_ws/install/setup.bash
fi

export ROS_DOMAIN_ID=30
export RMW_IMPLEMENTATION=rmw_fastrtps_cpp
export ROS_LOCALHOST_ONLY=0

export FASTRTPS_DEFAULT_PROFILES_FILE=$HOME/dev/orbbec_ws/config/fastdds_udp_only.xml
export FASTDDS_DEFAULT_PROFILES_FILE=$HOME/dev/orbbec_ws/config/fastdds_udp_only.xml
```

Make it executable:

```bash
chmod +x ~/dev/orbbec_ws/scripts/host_orbbec_env.sh
```

### Verify Environment Variables

After sourcing the script:

```bash
echo $ROS_DOMAIN_ID
echo $RMW_IMPLEMENTATION
echo $ROS_LOCALHOST_ONLY
echo $FASTRTPS_DEFAULT_PROFILES_FILE
echo $FASTDDS_DEFAULT_PROFILES_FILE
```

Expected values:

```bash
30
rmw_fastrtps_cpp
0
/home/<user>/dev/orbbec_ws/config/fastdds_udp_only.xml
/home/<user>/dev/orbbec_ws/config/fastdds_udp_only.xml
```

## Build

From host Ubuntu or from the Docker container:

```bash
cd ~/dev/orbbec_ws
source /opt/ros/humble/setup.bash
colcon build
```

After build:

```bash
source scripts/host_orbbec_env.sh
```

Check that Orbbec ROS2 packages are visible:

```bash
ros2 pkg list | grep -i orbbec
ros2 pkg prefix orbbec_camera
```

## Launch Orbbec Camera

After sourcing the environment:

```bash
cd ~/dev/orbbec_ws
source scripts/host_orbbec_env.sh
ros2 launch orbbec_camera gemini_336.launch.py
```

## DDS / Fast DDS Configuration

The workspace uses:

```bash
config/fastdds_udp_only.xml
```

Environment variables:

```bash
export FASTRTPS_DEFAULT_PROFILES_FILE=$HOME/dev/orbbec_ws/config/fastdds_udp_only.xml
export FASTDDS_DEFAULT_PROFILES_FILE=$HOME/dev/orbbec_ws/config/fastdds_udp_only.xml
```

This is used to keep ROS2 communication consistent between host and container.
